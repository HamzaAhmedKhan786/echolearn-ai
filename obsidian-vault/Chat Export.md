## Purpose

Chat Export lets the learner keep useful tutor conversations as a study record.

## Product Behavior

- Save questions and answers locally.
- Include document title and date.
- Export to PDF for sharing or revision.
- Keep private by default.

## Desktop

Desktop can use the browser print-to-PDF flow first.

## Mobile

Mobile should later use native share/export APIs.

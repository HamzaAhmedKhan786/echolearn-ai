## Purpose

Mobile Native Bridge connects Flutter UI to Android and iOS capabilities.

## Android

- Native TextToSpeech.
- Document picker for readable text files.
- Chunking after import.
- Encrypted API key storage with Android Keystore.
- App state persistence.

## iOS

- AVSpeechSynthesizer.
- UIDocumentPicker for readable text files.
- Chunking after import.
- API key storage with Keychain.
- App state persistence.

## Later

- Full mobile PDF/DOCX/EPUB parsing.
- Microphone question flow.
- Wake phrase or speaker-name activation.
- Phone QA for picker, storage, and voice behavior.

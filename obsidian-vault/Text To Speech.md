## Purpose

Text To Speech reads uploaded study documents aloud so the learner can listen, pause, and ask questions while studying.

## Desktop

- Windows native TTS is the default fallback.
- Piper is optional for higher-quality local voices.
- The user should be able to control speed and stop playback.
- The app should continue to work even when Piper is not configured.

## Mobile

- Android and iOS use native device voices first.
- Mobile TTS is lighter than shipping a large voice model.
- Reader controls include speech speed and stop.

## Purpose

Piper Voice is the optional desktop local voice path for higher-quality offline speech.

## Required Files

- `piper.exe`
- Voice model `.onnx`
- Matching voice config `.onnx.json`

## Example Path

```text
C:\Users\DELL\Downloads\setups\piper_windows_amd64\piper\piper.exe
```

## Product Rule

Piper should be optional. If it is missing or misconfigured, EchoLearn should fall back to Windows native TTS.

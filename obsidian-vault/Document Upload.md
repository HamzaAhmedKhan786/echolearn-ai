## Purpose

Document Upload is the first user action: the learner imports study material and EchoLearn turns it into readable, searchable chunks.

## Desktop Formats

- PDF
- DOCX
- EPUB
- TXT and Markdown
- CSV, JSON, HTML, XML

## Mobile Formats

- Native mobile picker currently reads text-style files.
- Full PDF/DOCX/EPUB parsing on phones is a later mobile parser task.

## Output

- Document title
- Extracted plain text
- Local chunks
- Topic matching input
- Reader and TTS input

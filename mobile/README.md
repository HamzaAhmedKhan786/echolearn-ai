# EchoLearn AI Mobile

Mobile application for EchoLearn AI.

## Stack

- Flutter
- Riverpod
- Android Native TTS
- iOS AVSpeech
- llama.cpp mobile runtime
- FAISS
- SQLite + SQLCipher

## Platforms

- Android
- iOS

## Responsibilities

- Mobile UI
- Offline document reader
- Local AI Q&A
- Study tools
- Native TTS integration
- Secure local storage
- Model download and updates

## Current Implementation

- `flutter_app/pubspec.yaml`
- `flutter_app/lib/main.dart`
- Material 3 navigation shell
- starter Library, Reader, AI Tutor, Study, and Settings screens

Native Android/iOS bridges are reserved but not implemented yet.
